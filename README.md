# 🎯 Millionaire Quiz Game

A Python-based command-line quiz game inspired by the famous "Who Wants to Be a Millionaire?" format. Players answer multiple-choice questions, earn virtual prize money, and continue climbing the prize ladder until they answer incorrectly.

## 🚀 Features

- Multiple-choice quiz questions
- Prize money progression system
- Instant answer validation
- Interactive command-line gameplay
- Easy to customize and expand
- Beginner-friendly Python project

## 🛠️ Built With

- Python 3

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/yourusername/millionaire-quiz-game.git
```

2. Navigate to the project folder:

```bash
cd millionaire-quiz-game
```

3. Run the game:

```bash
python main.py
```

## 🎮 How to Play

- Read each question carefully.
- Choose the correct option:
  - 1 = A
  - 2 = B
  - 3 = C
  - 4 = D
- Earn prize money for every correct answer.
- The game ends when you select an incorrect answer.
